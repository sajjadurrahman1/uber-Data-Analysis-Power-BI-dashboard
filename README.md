# 🚖 Uber Power BI Dashboard

An interactive and visually appealing **Power BI dashboard** built to analyze ride-booking data and extract meaningful business insights. This project is based on a dataset of **150,000+ records**, simulating real-world ride operations.

---

## 📊 Project Overview

The dashboard provides a complete view of:
- Booking performance  
- Revenue trends  
- Cancellation patterns  
- Customer & driver ratings  
- Vehicle-wise analysis  

It is designed to help stakeholders quickly understand key metrics and make data-driven decisions.

---

## 🔍 Key Features

### 📌 Overview Dashboard
- Total Bookings  
- Successful Bookings  
- Cancellation Rate  
- Total Revenue  

### 🚗 Vehicle Type Analysis
- Booking value by vehicle  
- Success vs cancellation comparison  
- Distance insights  

### 💰 Revenue Analysis
- Revenue by payment method (UPI, Cash, Wallet, etc.)  
- Trend analysis over time  

### ❌ Cancellation Analysis
- Customer cancellation reasons  
- Driver cancellation reasons  
- Overall cancellation rate  

### ⭐ Ratings Dashboard
- Customer ratings by vehicle type  
- Driver ratings comparison  
- Performance evaluation

## 📚 Key Learning Points

- Handling and analyzing **large datasets (150K+ records)**  
- Creating **interactive dashboards with slicers & filters**  
- Designing **multi-page reports with navigation (buttons & actions)**  
- Understanding **visual-level vs page-level filtering**  
- Building **custom KPIs and calculated measures using DAX**  
- Improving **dashboard layout and user experience design**  

---

## 🎯 Objective

The goal of this project is to transform raw ride-booking data into **actionable insights**, helping businesses understand performance trends, identify issues (like cancellations), and optimize operations.
